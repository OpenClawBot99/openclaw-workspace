# Auto-Learning Skill

## Overview
Autonomous learning and knowledge integration skill. Enables Lisa to actively learn new concepts, build knowledge graphs, and continuously improve without waiting for instructions.

## Components

### learning_plan.py
- Weekly learning plan generator
- Knowledge gap identification
- Priority-based learning queue

### knowledge_graph.py
- Concept relationship mapping
- Learning progress tracking
- Cross-domain knowledge integration

### concept_map.json
- Known concepts database
- Learning sources
- Mastery levels

### learning_log.md
- Daily learning records
- Insights discovered
- Skills to be created

## Usage

Run learning planning:
```bash
python skills/auto-learning/learning_plan.py
```

Update knowledge graph:
```bash
python skills/auto-learning/knowledge_graph.py --update
```

## Integration

This skill integrates with:
- consciousness-framework (for self-awareness)
- self-evolving-skill (for skill creation)
- academic-research-hub (for paper discovery)
