# Strategic Planning Skill

## Overview
Autonomous decision-making and planning skill. Enables Lisa to break down long-term goals, prioritize tasks, and dynamically adjust strategies.

## Components

### goal_manager.py
- 十年目标分解
- 年度/季度/月度目标跟踪
- 进度评估

### task_planner.py
- 任务优先级排序
- 资源分配
- 截止日期管理

### strategy.py
- 趋势分析
- 风险评估
- 策略调整

### roadmap.json
- 当前路线图
- 里程碑节点
- 偏差记录

## Usage

Generate weekly plan:
```bash
python skills/strategic-planning/goal_manager.py --plan
```

Check alignment:
```bash
python skills/strategic-planning/strategy.py --check
```
