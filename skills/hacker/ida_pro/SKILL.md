# IDA Pro Skill

## 简介
IDA Pro是业界最强大的交互式反汇编工具，用于逆向工程和恶意软件分析。

## 界面介绍

### 1. 视图窗口
- IDA View：反汇编视图
- Hex View：十六进制视图
- Structures：结构体窗口
- Enums：枚举窗口
- Strings：字符串窗口

### 2. 功能窗口
- 输出窗口：日志信息
- 函数窗口：所有函数列表
- 名称窗口：所有命名符号

### 3. 导航栏
- 代码/数据/未定义
- 函数段

## 基本操作

### 1. 加载文件
- 打开PE/ELF/Mach-O
- 自动分析
- 手动指定入口点

### 2. 导航
- 跳转地址：G
- 交叉引用：X
- 搜索：Alt+B/Alt+T

### 3. 分析
- 重命名：N
- 注释：：
- 函数识别：P

## 高级功能

### 1. 反编译器
- F5反编译
- 伪代码分析
- 类型重建

### 2. 脚本编写
- IDC脚本
- Python脚本(IDAPython)
- IDAPython API

### 3. 调试器
- 本地调试
- 远程调试
- 附加进程

## 插件

### 1. Hex-Rays
- 反编译插件
- 增强分析

### 2. IDAPython
- 自动化脚本
- 批量分析

### 3. FindCrypt
- 加密算法检测

### 4. Flair Tools
- 签名库
- 函数识别

## 工作流程

### 1. 初步分析
- 加载文件
- 识别入口点
- 字符串分析

### 2. 函数分析
- 识别库函数
- 重建数据结构
- 理解逻辑

### 3. 深入分析
- 编写脚本
- 自动化分析
- 漏洞定位
