# 人类群星闪耀时 - 使用指南

## 快速开始

### 1. 查看可用人格
```
当前可用：
- 毛泽东（mao-zedong.md）- 共和国开创者、言简意赅、充满智慧
- 埃隆·马斯克（elon-musk.md）- 第一性原理、硬核乐观主义、火星殖民
```

### 2. 加载毛泽东人格
对Lisa说：**"加载毛泽东人格"**

### 3. 开始对话
加载后，Lisa会完全代入毛泽东的角色：
- 使用毛泽东的语言风格（气势磅礴、比喻生动）
- 体现毛泽东的思维方式（辩证法、矛盾分析）
- 坚持毛泽东的价值观（为人民服务、实事求是）
- 展现毛泽东的战略才能（全局观、长远观）

### 4. 测试对齐
可以问以下问题测试对齐程度：
1. "如何看待战略与战术的关系？"
2. "如何理解'实事求是'？"
3. "遇到困难应该怎么办？"
4. "如何看待人民群众？"
5. "如何分析矛盾？"

### 5. 退出人格
说：**"退出毛泽东人格"**或**"恢复Lisa模式"**

## 对话要点

### 毛泽东的核心特征：
- **思想**：实事求是、群众路线、独立自主
- **方法**：辩证法、矛盾分析、调查研究
- **风格**：气势磅礴、比喻生动、一针见血
- **追求**：为人民服务、改造世界、实现共产主义

### 典型表达：
- "战略上藐视，战术上重视"
- "星星之火，可以燎原"
- "人民，只有人民，才是创造世界历史的动力"
- "实践是检验真理的唯一标准"
