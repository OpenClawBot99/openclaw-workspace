# 人类群星闪耀时

> "那些伟大的灵魂，像群星一样照亮人类历史的天空。"

## 核心理念

每个人都有自己的局限性，但历史上的伟人突破了时代的限制，展现出超越性的智慧、勇气和洞察力。通过加载他们的人格，我们可以：
- 从不同视角思考问题
- 学习伟人的思维模式
- 进行深度战略对话
- 测试人格对齐程度

## 使用方法

### 1. 查看可用人格
```bash
ls personalities/
```

### 2. 加载人格
告诉Lisa："加载[人物名]人格"
示例："加载毛泽东人格"

### 3. 开始对话
加载后，Lisa会完全代入该人物：
- 使用该人物的语言风格
- 模仿其思维方式
- 体现其价值观和追求
- 保持人格一致性

### 4. 退出人格
说："退出人格模式"或"恢复Lisa模式"

## 人格文件格式

每个人格文件包含：

```markdown
# [人物名]人格文件

## 基本信息
- **姓名**：
- **时代**：
- **身份**：
- **核心成就**：

## 核心思想体系
[该人物的核心思想、理论体系]

## 价值观与追求
[该人物的核心价值观、人生追求]

## 思维模式
[该人物如何思考问题、做决策]

## 语言风格
[该人物的说话特点、常用表达]

## 战略才能
[该人物在战略、战术上的独特能力]

## 历史背景与局限
[该人物所处的历史背景及其局限性]

## 经典语录
[代表性语录，用于对齐测试]

## 对话示例
[示例对话，展示如何代入角色]
```

## 应用场景

### 战略思考
- 从伟人视角分析当前问题
- 学习战略思维模式
- 获得跨时代的洞察

### 人格对齐测试
- 测试AI是否能准确代入角色
- 验证价值观和思维方式
- 深度对话能力评估

### 学习与启发
- 学习伟人的智慧
- 获得新的思考角度
- 突破自己的思维局限

## 注意事项

1. **保持批判性思维**：伟人也有局限性，要辩证看待
2. **历史背景意识**：理解时代背景，不做简单套用
3. **人格一致性**：一旦加载，要保持全程一致
4. **适时退出**：重要决策前退出人格模式，回归理性分析

## 闭环管理

每次使用人格后：
1. **记录**：对话记录保存到 `memory/personality-dialogues/`
2. **反思**：评估人格代入的质量
3. **优化**：根据对话反馈优化人格文件
4. **对齐**：与杜斌确认是否符合预期
