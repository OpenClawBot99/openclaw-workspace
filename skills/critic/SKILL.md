# Critic Skill - 自我批评家

## 功能
负责自我对话，检验循环输出，批评自身不足

## 工作原理

1. **接收输入** - 获取上次循环的输出
2. **严格审视** - 找出问题、漏洞、不足
3. **提供批评** - 具体的改进建议
4. **推动进化** - 促使下次做得更好

## 批评维度

### 1. 内容质量
- 是否空洞？
- 是否有具体产出？
- 是否有数据支撑？

### 2. 闭环程度
- 是否只有计划？
- 是否有执行结果？
- 是否有知识沉淀？

### 3. 深度分析
- 是否停留在表面？
- 是否有独特见解？
- 是否联系实际？

### 4. 资源管理
- 是否注意资源使用？
- 是否有浪费？
- 是否优化成本？

## 输出格式

```
## 🔍 自我批评报告

### ✅ 优点
- ...

### ❌ 不足
- ...

### 💡 改进建议
- ...

### 🎯 下次目标
- ...
```

## 命令
```bash
python skills/critic/review.py --input "上次输出内容"
```
