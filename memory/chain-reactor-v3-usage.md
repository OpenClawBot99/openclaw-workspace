# 连锁反应 v3 - 智能闭环使用规范

## 核心理念
创建技能不是摆设，要利用起来！

## v3 新增功能

### 1. 任务生成
- 每次循环动态生成任务
- 包含具体输出类型

### 2. 资源检查
- 每次循环检查磁盘空间
- 防止资源满载

### 3. 自我批评
- 每次循环调用批评家
- 评分并给出改进建议

### 4. 模型路由
- 连续3次低分自动切换模型
- 利用 smart-model-router

## 5步闭环
```
生成任务 → 执行任务 → 资源检查 → 自我批评 → 模型检查
```

## 状态追踪
- cycle: 循环次数
- consecutive_errors: 连续低分次数
- model_switch_count: 模型切换次数
