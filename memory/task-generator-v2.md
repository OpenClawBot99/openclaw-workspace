# 智能任务生成器 v2 - 小目标进化版

## 核心理念
积跬步以至千里！目标太大？拆解成小目标！

## 小目标示例

### docker（拆解成6步）
1. 安装Docker Desktop
2. 运行第一个容器hello-world
3. 理解docker ps命令
4. 学会docker pull
5. 编写简单Dockerfile
6. 构建第一个镜像

### linux（拆解成6步）
1. 打开PowerShell
2. 运行ls命令
3. 理解文件目录
4. 学习cd命令
5. 学习进程ps命令
6. 理解管道|

## 迭代机制
每次根据上次"药方"调整任务：
- 上次说"缺乏实践" → 选择可复现的小目标
- 上次说"缺乏资源" → 先检查资源
- 上次说"太简短" → 详细展开

## 目标大小
- tiny: 5分钟可完成
- small: 30分钟可完成
- medium: 1小时
- large: 超过1小时（避免）
