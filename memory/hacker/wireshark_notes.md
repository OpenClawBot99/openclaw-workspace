# Wireshark网络抓包分析学习笔记

## 什么是Wireshark?
Wireshark是世界上最流行的网络协议分析器，可以捕获和分析网络流量。

## 常用功能

### 1. 流量捕获
- 实时捕获网络包
- 支持多种接口
- 过滤特定流量

### 2. 协议解析
- 支持1000+协议
- 自动解码
- 层层封装分析

### 3. 过滤器
- BPF语法
- 显示过滤器
- 捕获过滤器

## 黑客实战应用

### 1. 网络扫描分析
- 分析Nmap扫描结果
- 识别开放端口
- 服务指纹识别

### 2. 密码嗅探
- HTTP明文密码
- FTP明文传输
- Telnet明文

### 3. 会话劫持
- 分析TCP会话
- 重放攻击
- ARP欺骗检测

### 4. 恶意流量检测
- 异常通信模式
- 后门通信
- C2流量识别

## 常用过滤表达式

### 1. 按协议过滤
```
tcp
udp
http
dns
```

### 2. 按IP地址过滤
```
ip.addr == 192.168.1.1
ip.src == 10.0.0.1
ip.dst == 8.8.8.8
```

### 3. 按端口过滤
```
tcp.port == 80
udp.port == 53
http.request.method == "POST"
```

### 4. 按内容过滤
```
http contains "password"
tcp contains "login"
```

## 高级技巧

### 1. 跟随TCP流
- 右键 → 跟随 → TCP流
- 查看完整会话

### 2. 专家信息
- 分析 → 专家信息
- 识别潜在问题

### 3. 统计功能
- 统计 → 协议分级
- 流量分布分析

## 防御措施

### 1. 使用加密
- HTTPS替代HTTP
- SSH替代Telnet
- SFTP替代FTP

### 2. 网络分段
- VLAN隔离
- 防火墙过滤

### 3. 入侵检测
- IDS/IPS部署
- 流量异常告警
