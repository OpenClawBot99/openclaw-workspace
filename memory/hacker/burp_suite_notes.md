# Burp Suite Web安全测试学习笔记

## 什么是Burp Suite?
Burp Suite是Web应用安全测试的集成平台，是最流行的Web渗透测试工具。

## 主要功能

### 1. Proxy(代理)
- 拦截HTTP/HTTPS流量
- 修改请求/响应
- 手动测试

### 2. Spider(爬虫)
- 自动爬取网站结构
- 发现隐藏页面
- 站点地图构建

### 3. Scanner(扫描器)
- 自动漏洞扫描
- SQL注入
- XSS
- CSRF

### 4. Intruder(入侵者)
- 暴力破解
- FUZZ测试
- 参数爆破

### 5. Repeater(中继器)
- 重放HTTP请求
- 修改参数测试
- 验证漏洞

### 6. Sequencer(序列器)
- 分析会话令牌
- 随机性检测

### 7. Decoder(解码器)
- URL编码/解码
- Base64
- 哈希生成
