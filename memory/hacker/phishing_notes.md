# 钓鱼攻击学习笔记

## 什么是钓鱼攻击?
钓鱼攻击是社会工程学攻击的一种，通过伪装成可信任实体诱骗受害者泄露敏感信息。

## 攻击类型

### 1. 钓鱼邮件
- 伪装银行、政府、公司
- 诱导点击恶意链接
- 附件恶意软件

### 2. 钓鱼网站
- 伪造登录页面
- 窃取账户密码
- 域名相似伪装

### 3. 短信钓鱼(Smishing)
- 伪装快递、银行
- 短链接诱导

### 4. 语音钓鱼(Vishing)
- 伪装客服电话
- 语音诱导

### 5. 水坑攻击
- 入侵常见网站
- 等待目标访问

## 攻击流程

### 1. 信息收集
- 社交媒体
- 公司网站
- 邮件地址

### 2. 伪装设计
- 伪造邮件/网站
- 制作钓鱼页面

### 3. 诱导执行
- 紧迫感制造
- 奖励诱惑
- 恐惧施压

### 4. 收获成果
- 账户信息
- 恶意软件安装

## 防御措施

### 1. 用户教育
- 识别钓鱼特征
- 不点击可疑链接
- 验证来源

### 2. 技术防护
- 邮件过滤
- 安全网关
- 双因素认证

### 3. 监测响应
- 异常检测
- 快速处置
