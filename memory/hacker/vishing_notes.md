# 假冒电话(Vishing)攻击学习笔记

## 什么是假冒电话?
假冒电话(Vishing)是语音钓鱼，通过电话诱骗受害者泄露敏感信息或转账。

## 攻击原理

### 1. 伪装身份
- 银行客服
- 政府部门
- IT支持

### 2. 心理操控
- 紧迫感
- 恐惧心理
- 奖励诱惑

### 3. 目标信息
- 银行账户
- 密码
- 验证码

## 攻击流程

### 1. 信息收集
- 社交媒体
- 公司网站
- 数据泄露

### 2. 脚本设计
- 话术准备
- 背景噪音
- 伪装工具

### 3. 实施攻击
- 拨打电话
- 心理诱导
- 获取信息

### 4. 收获成果
- 账户盗窃
- 资金转移
- 身份冒用

## 防御措施

### 1. 警惕陌生来电
- 不轻信
- 不提供信息

### 2. 验证身份
- 挂断回拨
- 官方渠道确认

### 3. 技术防护
- 来电显示
- 语音验证
