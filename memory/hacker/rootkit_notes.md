# Rootkit学习笔记

## 简介
Rootkit是一种恶意软件，设计用于隐藏自身并维持对系统的长期控制。

## 类型

### 1. 用户模式Rootkit
- 替换系统程序
- 隐藏进程/文件
- 键盘记录

### 2. 内核模式Rootkit
- 加载内核模块
- 挂钩系统调用
- 隐藏更深

### 3. 固件Rootkit
- BIOS/UEFI感染
- 网络卡固件
- 持久性最强

### 4. 虚拟化Rootkit
- 虚拟机监视器
- 最难检测
