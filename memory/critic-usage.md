# 自我批评家 v2 使用规范

## 核心原则
1. 每次循环后自动调用
2. 先肯定后否定
3. 有病要开药方
4. 闭环迭代

## 批评流程

### 1. 肯定（先找优点）
- 结构完整
- 有输出
- 有计划

### 2. 否定（后找不足）
- 内容简短
- 缺乏代码
- 缺乏资源管理

### 3. 原因分析
- 具体分析每个问题

### 4. 解决方案
- 每个问题对应一个药方

### 5. 下一步计划
- 具体可执行的任务

## 评分体系
- 基础分：50
- 优点：+10/个
- 严重问题：-15/个
- 严厉模式：-10
- 轻松模式：+5
