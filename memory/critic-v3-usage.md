# 自我批评家 v3 - 深度进化版使用规范

## 核心理念
- 先肯定后否定
- 有病要开药方
- 模板自我进化

## 批评维度（详细模板）

### ✅ 优点检查（4大类）
1. **结构** - 学习要点/落地措施/知识库/目标/下一步
2. **内容** - 代码/Skill/文档/笔记
3. **深度** - 原因分析/具体例子/数据支撑/对比
4. **行动** - 执行结果/验证过程/产出/测试

### ❌ 不足检查（5大类）
1. **空洞** - 简短、缺乏展开
2. **理论** - 只有理论、缺乏实践
3. **产出** - 无代码、无Skill
4. **资源** - 无资源管理
5. **闭环** - 无批评、无反思

### 💊 解决方案
每个问题都有对应药方

### 🎯 下一步计划
根据问题优先级安排具体行动

## 模板进化
- 每次迭代后分析历史
- 自动强化不足维度
- 动态更新检查项

## 评分体系
- 基础分：50
- 优点：+8/项
- 严重问题：-15/项
- 警告：-5/项
