# Lisa 任务输出格式标准 (2026-02-20)

## 格式标准

每次任务循环输出必须包含：

### 1. 学习要点 (Learning Points)
- 具体学到了什么
- 关键概念
- 认知提升

### 2. 落地措施 (Implementation)
- 具体做了什么
- 代码/笔记/项目
- 行动步骤

### 3. 知识库沉淀 (Knowledge Base)
- 文件路径
- 内容摘要

### 4. 目标与计划 (Goals)
- 底线目标 (Baseline)
- 挑战目标 (Stretch)

### 5. 下一步建议 (Next Steps)
- 明确的下一步
- 推荐方向

### 6. 新增功能 (New Features)
- 如果有改进
